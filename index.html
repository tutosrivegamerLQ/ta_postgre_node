acerca de

Transportadora Antina (TA) es una página la cual controla el sistema de envíos por parte de clientes, además de mercancías almacenadas por los mismos.

Este proyecto fue creado en base a un reto personal, ¿Cuál?, bueno es simple, anteriormente trabajabamos con archivos .json, mi trabajo es pasar la información de un .json a una base de datos PostgreSQL
